For instance, in Python we can sum to numbers using the + operator.

5.3 Integer division

For integers Python defines two other special operators: integer division (//), and remainder (%).

Tip 
print("9 % 3 ", 9 % 3)
print("14 % 2 ", 14 % 2)
print("14 % 3 ", 14 % 3)
print("9 // 3 ", 9 // 3)
print("14 //2 ", 14 //2)
print("14 % 3 ", 14 % 3)

5.4 Common unary operators

The operators that we have seen so far take two elements. For instance the add operator (+) takes two numbers and returns the sum of both. These operators are called binary because they take two items. But there are also unary operators; they only act over one element. The one most commonly used is the += operator.

There are other unary operators, like -=, *=, \= but they are not used as much.
